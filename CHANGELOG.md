# Changelog

All notable changes to the Student Management System project are documneted in this file.

This version follows a milestone-based versioning approach to track its development

## [v0.1.0]- Project Initialization

### Added
- Initial project structure
- Main menu interface
- Student class
- View all students
- Search students
- Array-based student storage

---

## [v0.2.0] - Student Management

### Added
- Edit student details
- Delete student records
- Unique Student ID generation
- Grade Management for six subjects
- View student grades

### Improved
- Input validation
- Console interface
- Program organization across multiple source files

---

## [v0.3.0] - Data Persistence

### Added
- Save student records to text files
- Load students on startup
- Automatic Student ID continuation
 
### Improved
- File handeling using C++ file streams
- Persistent data storage

---

## [v0.4.0] - Authentication & User Roles

### Added
- Admin login
- Student Login
- Password authentication
- Roled-based access control
- Separate Admin and Student menus
- Persistent account storage

### Improved
- Modular project architecture
- Separatin of authentication and student management logic

---
## [v.0.5.0] - Student Profiles 

### Added
- Student  email address
- Student phone number
- Year of student (defaults to Year 1)
- Student password change functionality

### Improved
- Student profile management
- Student profile editing
- Improved account management
